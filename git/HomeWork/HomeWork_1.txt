1) На GitHub.com перейдите на главную страницу репозитория.
Над списком файлов щелкните  Code .
2)Чтобы клонировать репозиторий с использованием ключа SSH выбрать его.
3)Откройте GIT Bash. Измените текущий рабочий каталог на расположение,
 где должен находиться клонированный каталог.Введите git clone и вставьте
 URL-адрес, скопированный ранее.


Задание - 2
(1) git init - Создать новый репо (выполняется только один раз,
создать новую базу данных для истории разработки)
		
(2) git add . - Взять на контроль все файлы проекта(сделать файл отслеж.)

(3) git commit -m 'какое-то описание' - Выполнить сохранение/фиксацию
(создать точку восстановления в проекте)

(4) Создать новый репо на GitHub
(открыть браузер и перейти в создание нового репозитория)

(5) Сделать "привязку" между локальным репо и репо на GitHub
		(делается один раз для нового репозитория)
		(скопировать комманду с GitHub)

(6) git push -u origin master(main) - Выгрузить локальный репозиторий на GitHub
	
Задание - 3
git fetch - для получения новых работ, выполненных другими людьми.
Команда получения данных из репозитория захватывает все новые ветви и теги удаленного
отслеживания, не сливая эти изменения с вашими собственными ветвями.
